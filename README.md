
# OpenCV 基础图像处理实验
基于 C++ 与 OpenCV 实现的**图像基础操作实验**，包含图片读取、信息打印、灰度转换、图像显示、像素访问、区域裁剪等核心功能，适合计算机视觉、数字图像处理课程入门练习。

## 实验环境
- 操作系统：Ubuntu Linux
- 开发工具：VS Code / C++ 编译器
- 依赖库：OpenCV 4.x
- 语言：C++

## 实验任务
本实验共完成 6 项基础图像处理任务：
1. 读取本地图片
2. 输出图像基本信息（宽、高、通道数、数据类型）
3. 显示原始图像
4. 彩色图像转灰度图像
5. 保存灰度图像到本地
6. 像素值读取 + 图像区域裁剪（ROI）

## 核心功能
- 图像读取与异常判断
- 图像基本信息打印
- 彩色图转灰度图
- 多窗口图像显示
- 指定坐标像素值读取
- 矩形区域（ROI）裁剪与保存

## 运行结果
- 控制台输出图像宽度、高度、通道数、数据类型
- 输出坐标 (100,100) 的 BGR 像素值
- 弹出原图、灰度图、裁剪图三个窗口
- 自动保存：gray_result.jpg、crop_result.jpg

## 使用方法
1. 将代码保存为 main.cpp
2. 修改 imread 中的图片路径
3. 编译并运行
4. 查看输出与生成的结果图片

## 代码结构
- type2str()：解析 OpenCV 矩阵数据类型
- main()：主流程，按顺序完成 6 个实验任务
